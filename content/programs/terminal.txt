Terminal commands

help

NTerminalC version 1.0.0 release (static-web)
These commands are defined internally. Type `help` to see this list.
Type `help name` to find out more about the function `name`

cd [dir | ]

    Changes the shell working directory

    Change the current directory to dir. The default value of dir is 
    the HOME shell variable.

    To navigate to the parent directory of the current working directory,
    `..` may be used. 

    Aside from navigation to the HOME directory, levels may only be traversed
    one at a time, from parent->child or from child->parent. Commands using `/`
    will be ignored.

help [name| ] 

    Display information about the built in terminal commands. 

    Displays a summary of the builtin command. If no command is
    specified, the list of potential help topics is printed.
    
list [dir| ]

    Prints the items in the desired directory.

    The default directory is the current working directory. 

    Aside from listing content in the HOME directory, content may only be listed in
    the current working directory.

look [value]

    For fans of Roberta, perhaps?

    At the beginning, look takes three possible values. Running look with any of these
    values gives you a small description and optional links...
     
    Further info about the reasoning behind these values can be gained by running `help look value`

    `Roberta`
    `terminal`
    `disk-scheduling-algorithms`

quit 

    Exits the currently running terminal program.

run [name] 

    Executes the program specified by name. Must be in the current working directory
    of the program in order to launch it successfully. Works for any installed program.

show [name]

    Displays the content of the file specified by name. Must be in the current working
    directory of said file in order to launch it successfully. Works for any .text extension.

tree [dir| ]

    Displays the structure of the subdirectories and files nested within the specified
    directory. By default uses the current working directory, but can take HOME as another
    value.
